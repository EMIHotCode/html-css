HTML (от англ. HyperText Markup Language) — это язык гипертекстовой разметки. Его используют для создания и структурирования веб-страниц. HTML помогает определять, как содержимое страницы должно отображаться в браузерах. С его помощью можно структурировать и форматировать текст на сайте, добавлять картинки, таблицы, списки.

CSS (англ. Cascading Style Sheets «каскадные таблицы стилей») — формальный язык декорирования и описания внешнего вида документа (веб-страницы), написанного с использованием языка разметки (чаще всего HTML или XHTML)

JavaScript (JS) — это высокоуровневый, интерпретируемый, мультипарадигменный язык программирования. Он используется в основном для создания интерактивных веб-страниц. JS поддерживают все популярные браузеры.
